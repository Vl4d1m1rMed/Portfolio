# Аналитик данных

Описание:

В репозитории представлены проекты курса Аналитик данных (Яндекс.Практикум).

Анализ данных: библиотеки pandas, numpy, scipy, statsmodels

Построение дашбордов: Tableau

Метрики юнит-экономики, когортный анализ

А/В-тестирование

Работа с гипотезами
